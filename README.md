# 🤖 Twitch AI Chat Bot

[![Python 3.8+](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Twitch](https://img.shields.io/badge/Twitch-IRC-purple.svg)](https://dev.twitch.tv/docs/irc)

Бот для Twitch чата, который отвечает на сообщения с помощью нейросети (OpenAI, Anthropic, локальные LLM и другие).

## ✨ Возможности

- 🔌 Подключение к любому Twitch каналу
- 🧠 Ответы на упоминания через ИИ
- 🎯 Игнорирование других ботов
- 🔄 Автоматическое переподключение при обрыве
- 📊 Статистика сообщений
- ⚡ Низкая задержка
- 🔒 Безопасное хранение ключей
